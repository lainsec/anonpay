# Transaction platform full written in python3

_**How to install:**_
```
git clone https://github.com/lainsec/anonpay
cd anonpay
pip install -r requirements.txt
```

_**How to use:**_
```
python3 app.py/
```

# FEATURES
> - _1. Message handeling_
> - _2. Responsive html/css_
> - _3. Create custom transactions_
