# Transaction platform full written in python3

_**How to install:**_
```
git clone https://github.com/lainsec/anonpay
cd anonpay
pip install -r requirements.txt
```

_**How to use:**_
```
python3 app.py/
```

_**Features:**_
> - 1. _Message handeling_
> - 2. _Responsive html/css_
> - 3. _Create custom transactions_
